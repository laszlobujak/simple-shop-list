# Képek mappája

## Struktúra

```
/images
  /hero         - Hero szekció képei
  /sections     - Különböző landing page szekciók képei
```

## Szükséges képek

### Hero szekció
- **hero-jewelry.jpg** (1920x1080 vagy nagyobb)
  - Keresési kulcsszavak: "luxury gold jewelry watches elegant"
  - Ajánlott: Unsplash, Pexels, Pixabay

### AI technológia szekció
- **ai-appraisal.jpg** (800x600 vagy nagyobb)
  - Keresési kulcsszavak: "smartphone technology app professional"

### Piactér szekció
- **marketplace.jpg** (800x600 vagy nagyobb)
  - Keresési kulcsszavak: "luxury marketplace jewelry collection"

## Képforrások (ingyenes, kereskedelmi használatra is)

1. **Unsplash** - https://unsplash.com
2. **Pexels** - https://pexels.com
3. **Pixabay** - https://pixabay.com

## Használat

```tsx

```
